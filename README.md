### Проект: Игры разума
#### Системные требования:

* python версии 3.6 или выше
* Установленный менеджер пакетов poetry
* Unix-система
***
Запуск:
1. Склонируйте репозиторий
2. Введите команду: <br>
`make build`
3. Затем 
`make package-install`
<br>
